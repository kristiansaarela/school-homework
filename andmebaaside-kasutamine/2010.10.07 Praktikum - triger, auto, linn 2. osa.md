AUTODE ja LINNADE trigeri ülesande jätk:



1.Lisa autode tabelisse väljad:

L_O_kuup (date) – liiklusõnnetuse kuupäev,

L_O_osaline (set) – kas auto oli süüdi või mitte (valik ‚’kannataja’, 						’tekitaja’),

K_koef (float) – liikluskindlustuse koefitsient.



mysql> alter table auto add column l_o_kuup date, add column l_o_osalone set('kannataja', 'tekitaja'), add column k_koef float(2,1);

Query OK, 0 rows affected (0.01 sec)

Records: 0  Duplicates: 0  Warnings: 0



2.Lisa autode tabelisse kõikidele autodele liikluskindlustuse koefitsiendid.

mysql> insert into auto values ('', 'bemm', '9187349', 2000, 1,'' ,'' , 1), ('', 'oopel', '24252', 2005, 2,'' ,'' , 1);



3.Täiusta tabeli AUTO muutmise (update) trigerit selliselt, et liiklusõnnetuse kuupäeva lisamisel suurendatakse auto kindlustuskoefitsienti 1,5 korda nende autode korral, kes olid õnnetuses süüdi.

mysql> create trigger auto_onnetus before update on auto for each row begin if OLD.l_o_kuup != NEW.l_o_kuup then set NEW.k_koef = OLD.k_koef*1.5; end if; end;//



4.Lisa mõnele autole AUTODE tabelis liiklusõnnetuse kuupäevad koos õnnetuses osaleja valikuga (kannataja, tekitaja).

mysql> update auto set l_o_kuup = '2010-10-14' where auto_id = 2;

Query OK, 1 row affected (0.00 sec)

Rows matched: 1  Changed: 1  Warnings: 0
