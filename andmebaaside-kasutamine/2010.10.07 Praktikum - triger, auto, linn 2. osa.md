1. Lisa autode tabelisse väljad:
	1. `L_O_kuup` (date) – liiklusõnnetuse kuupäev
	2. `L_O_osaline` (set) – kas auto oli süüdi või mitte (valik‚ ’kannataja’, ’tekitaja’)
	3. `K_koef` (float) – liikluskindlustuse koefitsient

	```sql
	ALTER TABLE auto
	ADD COLUMN l_o_kuup DATE,
	ADD COLUMN l_o_osalone SET('kannataja', 'tekitaja'),
	ADD COLUMN k_koef FLOAT(2,1);
	```

3. Täiusta tabeli `AUTO` muutmise (update) triggerit selliselt, et liiklusõnnetuse kuupäeva lisamisel suurendatakse auto kindlustuskoefitsienti 1,5 korda nende autode korral, kes olid õnnetuses süüdi.

	```sql
	DELIMITER //

	CREATE TRIGGER auto_onnetus
	BEFORE UPDATE ON auto
	FOR EACH ROW
	BEGIN
		IF OLD.l_o_kuup != NEW.l_o_kuup THEN
			SET NEW.k_koef = OLD.k_koef * 1.5;
		END IF;
	end//

	DELIMITER ;
	```

	> Query OK, 0 rows affected (0.01 sec)
