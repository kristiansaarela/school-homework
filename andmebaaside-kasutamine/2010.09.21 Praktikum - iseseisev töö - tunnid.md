# Ülesanne

On antud tabel, milles on üks veerg, kus hoitakse tunde täisarvudena. Sellest tabelist oleks vaja saada selline tulemus, mis näitaks kui palju ette antud tundidest saab nädalaid, päevi ja tunde.

nt.

|Tunnid|Nadalad|Paevad|Tunnid|
|------|-------|------|------|
|     5|      0|     0|     5|
|    55|      0|     2|     7|
|   125|      0|     5|     5|
|  1225|      7|     2|     1|
|   555|      3|     2|     3|

```sql
CREATE TABLE tunnid (tunnid INT);

INSERT INTO tunnid
	(tunnid)
VALUES (5), (55), (125), (1225), (555);

SELECT
	tunnid,
	FLOOR(tunnid / 168) AS nadalad,
	FLOOR((tunnid / 168 - FLOOR(tunnid / 168)) * 168 / 24) AS paevad,
	(tunnid - (FLOOR((tunnid / 168 - FLOOR(tunnid / 168)) * 168 / 24) * 24)) - FLOOR(tunnid / 168) * 168 AS tunnid
FROM
	tunnid;
```
