# Ülesanded trigerite kohta.

1. Loo tabel `AUTO` järgmiste väljadega:
	1. `auto_id` - täisarv väli peab olema täidetud primaarvõtme väli
	2. `mark` - fikseerimata pikkusega tekst sümbolite arv vali ise
	3. `RegNR` - fikseeritud pikkusega 9 sümbolit
	4. `aasta` - year
	5. `RegPiirk` - täisarv seotud linnade tabeliga (`LINNAD(Linn_ID)`)

	Kui linnade tabelit ei ole siis saad vastavad käsud ja andmed failist `LINN.txt`

	```sql
	CREATE TABLE auto (
		auto_id INT NOT NULL PRIMARY KEY AUTO_INCREMENT,
		mark VARCHAR(50) NOT NULL,
		regnr VARCHAR(9) NOT NULL,
		aasta YEAR,
		regpiirk INT NOT NULL,
		FOREIGN KEY (regpiirk) REFERENCES LINNAD(LinnID)
	);
	```
	> Query OK, 0 rows affected (0.00 sec)

2. Koosta tabel `LOGI` autoregistris toimuvate muudatuste logimiseks.
	Väljad (`id`, `auto_id`, `toiming`, `aeg`, `mark`, `RegNR`, `aasta`, `RegPiirk`).

	```sql
	CREATE TABLE logi (
		id INT NOT NULL PRIMARY KEY AUTO_INCREMENT,
		auto_id INT NOT NULL,
		toiming VARCHAR(50) NOT NULL,
		aeg DATETIME NOT NULL,
		mark VARCHAR(50) NOT NULL,
		regnr VARCHAR(9) NOT NULL,
		aasta YEAR NOT NULL,
		regpiirk INT NOT NULL
	);
	```
	> Query OK, 0 rows affected (0.01 sec)

3. Loo päästikud lisamiste, kustutamiste, muutmiste logimiseks. Mõtle, mida ja millistesse tulpadesse on kindlasti vajalik kirjutada, et logi järgi oleks võimalik vana seis taastada.

	```sql
	CREATE TRIGGER
		lisamine
	AFTER INSERT ON AUTO
	FOR EACH ROW
	BEGIN
		INSERT INTO
			logi
		VALUES
			('', NEW.auto_id, 'lisamine', NOW(), NEW.mark, NEW.regnr, NEW.aasta, NEW.regpiirk);
	END;;
	```

	```sql
	INSERT INTO
		auto
	VALUES
		('', 'opel', '846548643', '2010', '5');
	```

	SELECT * FROM logi;

4. Tee päring, mis näitab etteantud `auto_id` korral, milliseid toiminguid selle autoga on tehtud ajas vanematest sündmustest uuemateni. Päringu tulemus võiks olla loetav (lisa sobivaid sõnu).

	```sql
	SELECT
		CONCAT('auto mille id number on ', auto_id, ' on tehtud ', toiming) AS toiming
	FROM
		logi
	WHERE
		auto_id = 1;
	```

	| toiming                                         |
	|-------------------------------------------------|
	| auto mille id number on 1 on tehtud lisamine    |
	| auto mille id number on 1 on tehtud muutmine    |
	| auto mille id number on 1 on tehtud kustutamine |
