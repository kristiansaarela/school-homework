1. Leia kui palju erinevad laste pikkused keskmisest pikkusest. (Väljasta lapse nimi, pikkus, keskmine pikkus ja erinevus keskmisest pikkusest)

	```SQL
	SELECT
		L_nimi,
		Synnipikkus,
		(
			SELECT AVG(Synnipikkus)
			FROM SYNNID
		) AS keskmine_pikkus,
		(
			SELECT sub.Synnipikkus - keskmine_pikkus
			FROM SYNNID AS sub
			WHERE
				sub.L_nimi = s.L_nimi AND
				sub.Synnipikkus = s.Synnipikkus
		) AS erinevus
	FROM SYNNID AS s
	ORDER BY erinevus;
	```

6. Väljasta laste sünnikaalud järgmisel kujul (nt Juku 3kg 410grammi).

	```sql

		CONCAT_WS('', L_nimi, ' ' , (
			SELECT LEFT(Synnikaal, 1) AS kilo
			FROM SYNNID
			WHERE
				L_Nimi = s.L_nimi AND
				Synnikaal = s.Synnikaal

			SELECT RIGHT(Synnikaal, 3)
			FROM SYNNID
			WHERE
				L_nimi = s.L_nimi AND
	```
