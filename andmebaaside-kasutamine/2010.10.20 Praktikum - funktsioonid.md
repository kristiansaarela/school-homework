# Ülesanded funktsioonide kohta

Vaata [veebilehelt](http://www.java2s.com/) Procedure Function näidet 11.3.5

1. Loo funktsioon, mis etteantud reaalarvu (3 kohta peale koma) ümardab reaalarvuks, millel on 2 kohta peale koma.

	```sql
	DELIMITER //

	CREATE FUNCTION kaks(nr DECIMAL(10,3))
	RETURNS DECIMAL(10,2)
	BEGIN
		RETURN ROUND(nr, 2);
	END//
	```

	> Query OK, 0 rows affected (0.01 sec)

	```sql
	SELECT KAKS(548411.333)//
	```

	| kaks(548411.333) |
	|------------------|
	|        548411.33 |

	> 1 row in set (0.01 sec)

2. Muuda eelnevat näidet, et see väljastaks tabelist `SYNNID`.`ema_id` ja selle esinemiste arvu tabelis.

	```sql
	CREATE FUNCTION synn(emaid INT)
	RETURNS INT
	READS SQL DATA
	BEGIN
		DECLARE id INT;

		SELECT DISTINCT COUNT(*) INTO id
		FROM SYNNID
		WHERE Ema_id = emaid;

		RETURN id;
	END//
	```

	```sql
	SELECT
		ema_id,
		synn(ema_id)
	FROM SYNNID
	GROUP BY ema_id//
	```

	| ema_id | synn(ema_id) |
	|--------|--------------|
	|      1 |            1 |
	|      2 |            1 |
	|      3 |            1 |
	|      4 |            1 |
	|    ... |          ... |
	|     50 |            2 |

	> 50 rows in set (0.00 sec)

3. Kasutades näidet 11.1.4 loo funktsioon, mis leiab mitu last on igal kuupäeval sündinud.Tulemuse väljastamiseks kasuta select-päringut.

	```sql
	CREATE FUNCTION laps(sy TIME)
	RETURNS INT
	READS SQL DATA
	BEGIN
		DECLARE kooku INT;

		SELECT COUNT(*) INTO kooku
		FROM SYNNID
		WHERE Synniaeg = sy;

		return (kooku);
	end//
	```

	> Query OK, 0 rows affected (0.01 sec)

	```sql
	SELECT
		Synniaeg,
		laps(Synniaeg)
	FROM SYNNID
	GROUP BY Synniaeg//
	```

	| Synniaeg | laps(Synniaeg) |
	|----------|----------------|
	| 00:12:00 |              1 |
	| 00:15:00 |              1 |
	| 00:53:00 |              1 |
	| 01:13:00 |              1 |
	| 01:27:00 |              1 |

	> 52 rows in set (0.01 sec)

4. Loo funktsion, mis etteantud kuupäeva (nt 2008-11-25) muudab eestipäraseks (nt 25.11.2008). Kasuta MySql kuupäeva funktsiooni date_format oma funktsiooni tegemisel. Tulemuse väljastamiseks kasuta select-päringut.

	```sql
	CREATE FUNCTION eestiparaseks(aeg DATE)
	RETURNS VARCHAR(20)
	BEGIN
		RETURN DATE_FORMAT(aeg, '%e.%c.%Y');
	END//
	```

	> Query OK, 0 rows affected (0.00 sec)

	```sql
	SELECT eestiparaseks('2010-12-25')//
	```

	| eestiparaseks('2010-12-25') |
	|-----------------------------|
	| 25.12.2010                  |

	> 1 row in set (0.00 sec)
